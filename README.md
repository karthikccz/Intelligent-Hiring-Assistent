🧠 TalentScout – AI Hiring Assistant

An intelligent AI-powered technical screening assistant built using Streamlit + Google Gemini API.

This system conducts structured interviews by:

Collecting candidate details

Generating beginner-to-medium level technical questions

Managing timed responses

Recording answers

Generating downloadable interview reports

Supporting structured post-interview Q&A

🚀 Features
🎯 Intelligent Question Generation

Generates exactly 10 beginner-to-medium level questions

Technology-specific

Experience-aware

Dynamic (varies each session)

⏳ Timed Interview System

2 minutes 30 seconds per question

Live countdown timer

Auto move to next question on timeout

💬 Real-Time Chat Support

ChatGPT-style interface

User messages on right

Assistant responses on left

“Generating answer…” indicator

Strict question-number-only responses

📊 Answer Recording

Stores candidate responses

Records timeout answers

Saves interview history to JSON

📄 PDF Report Generation

Candidate details

All 10 questions

All responses

Downloadable structured report

🔒 Controlled Scope

Only beginner–medium questions

No advanced/system design questions

No unrelated queries allowed

Exit Portal disables session
